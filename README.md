# Precision-and-Recall

본 게시글은 인식, 분류, 검출, 검색 등의 작업 성능 평가시 활용되는 척도인 **Precision(정밀도)** 과 **Recall(재현율)** 에 대한 이해를 돕고자 작성되었습니다.

우선 **Precision** 과 **Recall** 을 설명하기에 앞서 우리가 해결해야할 문제를 정의하고 문제를 해결하기 위한 방법에 대한 성능 평가 척도로 **Precision** 과 **Recall** 을 실제 문제에 적용하는 형태로 설명을 드리겠습니다.
